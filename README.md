### Simple Java MVC with embedded H2 demo (console client)

#### How to run
Use maven to build, then run main class Main

```
mvn install
mvn package
java -jar target/demo-1-jar-with-dependencies.jar
```

#### JS + Ajax demo

To open in Chrome on Mac OS, disable CORS policy:
```open -n -a /Applications/Google\ Chrome.app/Contents/MacOS/Google\ Chrome --args --user-data-dir=/tmp/chrome_dev_test --disable-web-security```

Then run ./lab2_Katalnikov.html
